> :hammer: Find emails in plain text file

## Install

```
$ npm install -g find-emails-in-string-cli
```

## CLI

```
$ find-emails /path/to/text-file.txt
admin@example.org, webmaster@example.org, root@example.org
```

## Unit tests

$ npm test
